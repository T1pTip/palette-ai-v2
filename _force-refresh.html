💡 אם זה לא עוזר (iOS Safari אגרסיבי במיוחד)

iPhone:
  1. סגור לגמרי את Safari (App Switcher → swipe up על ה-Safari)
  2. אם התקנת PWA על המסך הראשי — מחק אותו (לחץ-ארוך → Delete App)
  3. Settings → Safari → Advanced → Website Data → חפש t1ptip → swipe → Delete
  4. פתח Safari מחדש ולך ל-https://t1ptip.github.io/palette-ai-v2/

Android Chrome:
  1. Chrome → 3 נקודות → Settings → Site settings → All sites
  2. חפש t1ptip.github.io → לחץ עליו → "Clear & reset"
  3. פתח טאב חדש ולך לאתר
